Cash Collection
This module will enable to post multiple payments simultanously. It is very useful for field team to post and reconcile bulk payments on daily basis.

Features

  • create collection document for both customers and vendors.
  • Auto generate payment documents for each invoice.
  • Auto payment matching with invoice.
  • Print statement.

Create Cash Collection for customer or vendor

Select payment type and partner type
Select Payment Journal (cash or bank)

Generate Payments

generate payment respective payment document for each invoice
Allow to enter less amount than due amount

Print Statement

user can reconcile daily collection with statement
