Hide / Invisible Any Menu Sub-Menu From Any User


Work With Both Community And Enterprise Version.

If you want to hide unwanted menu, sub-menu for particular user then just install this app.

Go to setting --> Users --> Open Users --> Hide Specific Menu Page --> Select any Menu, Sub menu that you want ot hide for this user. --> Just save and refresh/restart Odoo and enjoy.

Key features:
  • Easy To Use
  • User Wise Setting
  • Hide Any Menu, Sub-Menu
  • Setting Can Applied For Administrator Too.
Suppose You Want To Hide Some Menus Like : Main Menu Project, Purchases, Sub Menu General Settings From Setting, Catalog From Sale, Sales Channels From Sales --> Reporting.
Select Menus Save And Refresh Page.
Menu General Settings is Invisible.
Before Applied Hide Menu Main Menu(App) Purchases And Project Are Visible
After Refresh Main Menu(App) Purchases And Project Are Invisible
Before Applied Hide Menu Catalog And Sub Menu Sales Channels Are Visible
After Refresh Menu Catalog And Sub Menu Sales Channels Are Invisible
