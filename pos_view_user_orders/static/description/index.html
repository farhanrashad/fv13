POS Display Orders User-wise

By default, in Odoo, all the orders of Point of Sale are visible to every user. But sometimes we wish that there should be a restriction in Orders, in terms of visibility, hence we have introduced our module Odoo POS Display Orders User-wise, using which admin can restrict the user to view the Orders of his POS only. So accordingly the particular user can view only his POS Orders.

Features
  • Admin can restict users to view only his POS Orders
  • POS Users would have the access to view only their POS Orders.
