Create Claim

Create claim based on sale order. Order lines is auto populate based on sale order.
Here, user can specify the location for return products.
System will not allow to update claim data after confimation of claim.
User can select return type as Exchange , Credit Return or Repair.

Create In-shipment

User can specify how many quantity or how many products they want to return.
When Return Type is 'Repair' or 'Exchange' user can create In-Shipment from Customer Location to Damaged Location.

In-Shipment

By click on 'Receive Product' button use can show In-shipment from customer to damaged location.

Create New Product Delivery

After creation of In-shipment it show you 'Deliver Product' button for transfer New Fresh product from stock to customer when Return Type is 'Repair' or 'Exchange'.
When user Transfer this delivery order after claim will move on Done state Return Type is 'Repair' or 'Exchange'.

Create Refund Invoice

User can create 'Refund Invoice' after creation of in-shipment when Return Type is 'Credit Return'.
When user done payment of 'Refund Invoice' after claim will move on Done state Return Type is 'Credit Return'.
