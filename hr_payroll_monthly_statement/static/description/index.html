HR Employees Payroll Monthly Statement

This module allow HR to print pdf report for payroll statement by salary rule and employee.

Menus:
 Payroll/Report/Payroll Monthly Statement

Payroll Monthly Statement Wizard


By clicking this Export Excel button it will export monthly payroll statement for selected employees. Note that employees on wizard is option if you do not select employees in wizard then module will print for all employees in system where payslips have been generated.

Output - Excel Report
