POS Order Notes

Allows the seller to add notes to individual orderlines as well as the complete order in POS

This Extension Allows

  • Seller can add notes for each orderline in POS.
  • Seller can add a note for the complete order.
  • Seller can activate or deactivate the order note feature.
  • Seller can print the order note on the receipt if he wants.
  • Quick and Easy Installation.
